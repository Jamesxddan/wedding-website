# ui-styling v1.0.0

shadcn/ui + Tailwind CSS component implementation skill.

## Stack

- **shadcn/ui** — Radix UI primitives with accessible, unstyled components
- **Tailwind CSS** — Utility-first CSS framework
- Mobile-first, accessible by default
- Dark mode via CSS variables

## Setup

```bash
npx shadcn@latest init
```

## Add Components

```bash
npx shadcn@latest add button card dialog form
```

## Principles

- Mobile-first responsive design
- Accessible (WCAG 2.1 AA)
- Dark mode support via CSS custom properties
- Composable, unstyled primitives

## Usage

Use when implementing UI components with shadcn/ui and Tailwind. Pairs with `ui-ux-pro-max` for design direction and `banner-design` for marketing assets.
